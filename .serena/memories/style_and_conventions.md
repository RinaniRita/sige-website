# Style and Conventions

- **Language**: TypeScript
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS 4.x
- **Components**: Radix UI, Shadcn/ui (implied by components.json and structure)
- **State Management**: React Hooks, React Hook Form
- **Validation**: Zod
- **Icons**: Lucide React
- **Animations**: tw-animate-css, Framer Motion (often used with Shadcn, though not in package.json yet)
- **Data Fetching**: Next.js fetch/server components or specialized clients in `lib/`.
